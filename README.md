# users-android

Application sample using the new MVI pattern (Model-View-Intent)

* Dependency Injection using Dagger 2, 
* Model View Intent pattern
* RxJava (RxKotlin)
* Room database,
* Retrofit, 
* viewModels,
* Instrumentation testing with Espresso
* jUnit testing business logic controllers.
* Jetpack navigation
* Safe Args
* Others...
